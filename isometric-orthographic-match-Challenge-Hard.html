Setup

Set up the round

Students enter their details, choose a game mode, and then match the orthographic drawings to each isometric block. Correct choices stay locked in place. Wrong choices count against the score in sprint mode.

How students play

  1. Pick the current target block from A–F, or stay with the automatically selected one.
  2. Click up to three orthographic cards that belong to that block.
  3. Use Check selection to lock correct views and release wrong ones.
  4. Finish all six blocks for a sheet, or keep going through new sheets in sprint mode.

Dashed lines show hidden detail. A short success message appears after each block is fully matched.

This page runs entirely in the browser, so it can be saved as a single HTML file and shared with students.
